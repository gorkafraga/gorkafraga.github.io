# Analysis

## Summary statistics
Summarize, convert to wide format and computes percentage

```{r }
data_sum = data %>%
  group_by(participant, familiarity, condition, response) %>% 
  count(response) %>% 
  spread(response, n) # converts to wide format


# add percentage correct column
data_sum$percent = (data_sum$hit/40)*100

# render table 
DT::datatable(data_sum, options = list(dom = 't', 
                                   scrollX = TRUE, 
                                   scrollY= "400px",
                                   pageLength = 999999))

```
